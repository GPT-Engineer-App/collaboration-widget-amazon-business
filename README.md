## Setup

```sh
git clone https://github.com/GPT-Engineer-App/collaboration-widget-amazon-business.git
cd collaboration-widget-amazon-business
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
